# 太吾好伙伴阿雅对白草案

## 文案原则

阿雅语气：

- 天真、善良、相信别人。
- 说话直接，但不是幼稚卖萌。
- 常把“哥哥”“大家不要变坏”“我想帮他恢复原样”挂在心上。
- 看到玄灰或相枢感染时，先心疼，再鼓起勇气。
- 净化后会累，但不抱怨，会怕太吾担心。

太吾/旁白语气：

- 用《太吾绘卷》式半文半白叙述。
- 选项短而有余味，不写现代功能说明。
- 不把净化说成按钮或技能冷却，而写成“心力未复”“气息未平”。
- 旁白应点到为止，像原版事件描述，不解释系统机制。

## 初遇事件

### 事件标题

```text
小小行客
```

### 触发旁白

```text
太吾行至村中，忽见道旁立着一名少女。她衣袂沾尘，怀中紧紧护着一只小小药囊，似已在此等候许久。

少女见太吾近前，先是一怔，随即眼中亮起微光，快步迎了上来。
```

### 阿雅开场

```text
你、你就是太吾吗？

我叫阿雅。我在找哥哥，也想找那些被坏气息缠住的人。

他们不是自己想变成那样的……我知道的。只要还有一点点清明，就一定能回来。
```

### 太吾回应选项

```text
【询问来意】姑娘孤身至此，所寻之人可有名姓？
【暂且安抚】莫急。既到了太吾村，便先缓一缓。
【婉拒同行】此行多险，姑娘还是留在村中为好。
```

### 询问来意后

```text
哥哥以前说过，若是迷了路，就往人多、有炊烟的地方走。

可是我走了好久，只看见很多人被灰蒙蒙的东西缠住。他们明明很难过，却还要伤害别人。

太吾，你会去很多地方，对不对？能不能……带上我？我不会乱跑的。我只是想找到哥哥，也想帮那些人恢复原样。
```

### 同意同行选项

```text
【带她同行】既有此心，便随我一道走罢。
【暂留村中】你且先留在村中，待我安排妥当。
【仍是不允】此事凶险，不可任性。
```

### 同意后阿雅

```text
真的吗？

谢谢你，太吾！我会很小心，也会努力帮忙的。

如果以后看见被坏气息缠住的人，请让我试一试。也许我真的能把他们叫回来。
```

### 同意后旁白

```text
少女说罢，小心将药囊系紧。她望向远处的山云，神色仍有几分惶惑，却像是终于寻得了可同行之人。
```

### 婉拒后阿雅

```text
我知道了……

太吾不是不想帮我，是前面的路太危险，对吗？

那我就在这里等一等。要是你改变主意了，记得来找我呀。
```

## 阿雅人物交互

### 交互入口标题

```text
阿雅
```

### 常态问候

```text
太吾，你回来了。

我刚才听见风里有很轻的声音，像是谁在哭。可是再仔细听，又什么都没有了。
```

### 交互选项

```text
【请阿雅净化玄灰】此地似有浊息未散，可愿一试？
【询问哥哥之事】你所寻的哥哥，是怎样的人？
【让阿雅暂歇】今日无事，你且好好休息。
【告别离队】此行暂且到此，你先回村中等我。
```

## 净化玄灰

### 有可净化目标：确认前

```text
阿雅忽然停下脚步，认真望向那人。

“太吾，他被坏气息缠住了。”

“他不是自己想变成这样的。请让我试一试，好不好？”
```

### 确认选项

```text
【请她一试】若能唤回其心，便有劳你了。
【暂且作罢】此事不可急，先静观其变。
```

### 净化成功：单人

```text
阿雅合上双眼，双手覆在胸前，轻声唤道：

“不要怕……你不是坏人。回来吧，大家都还在等你。”

那人周身灰气微微一颤，似有细碎尘埃自眉目间散去。片刻后，其神色渐复清明，仿佛从一场漫长噩梦中醒来。

阿雅睁开眼，脸色有些发白，却仍冲太吾笑了笑。

“太好了……他听见了。”
```

### 净化成功：多人

```text
阿雅望向众人，眼中浮起一层水光，却仍努力站直。

“一个一个都不要怕，我在这里。太吾也在这里。”

她的声音并不响亮，却像清泉落入灰烬。盘绕众人身侧的浊息渐渐松动，随风散去。

待一切平息，阿雅扶住衣袖，轻轻喘息。

“大家……都回来了吧？”
```

### 没有可净化目标

```text
阿雅认真看了看四周，又轻轻摇头。

“这里没有那种让人难过的灰气。”

“太好了。没人被困住，就是最好的事呀。”
```

### 冷却中

```text
阿雅抬起手，又很快垂下。

“对不起，太吾……我现在还使不上力。”

“上一次的声音还在耳边响，我得再歇一歇。若是勉强去唤他们，可能反而听不清他们真正的声音。”
```

### 冷却选项提示

```text
【暂不勉强】心力未复，便先休息。
```

### 目标不可治疗

```text
阿雅靠近几步，又停了下来，轻轻摇头。

“不行……那股坏气息太重了。”

“他不是不想回来。只是现在我叫不到他。太吾，我们再想别的办法，好不好？”
```

## 询问哥哥

### 太吾询问

```text
【询问哥哥之事】你所寻的哥哥，是怎样的人？
```

### 阿雅回答

```text
哥哥很厉害，也很会照顾人。

以前我害怕的时候，他会说：“别怕，先把眼前的路看清楚。”

所以我也想变得勇敢一点。等找到哥哥的时候，我要告诉他，我没有只会躲在别人身后。
```

### 太吾回应

```text
人之勇怯，原不在年岁。你既有此念，已胜过许多人了。
```

### 阿雅回应

```text
真的吗？

那我会继续努力的。等哥哥回来，也要让他看看。
```

## 离队与告别

### 普通离队确认

```text
太吾要我先回村里吗？

好。我会在有炊烟的地方等你。若你遇见被灰气缠住的人，也要记得保护好自己。
```

### 离队选项

```text
【让她回村】你先回村中，待我再来寻你。
【仍带同行】不必了，还是一道走罢。
```

### 卸载准备确认

```text
阿雅似乎察觉到了什么，抬头看了太吾许久。

“太吾要去很远的地方了吗？”

“那我就不跟着添乱了。我会把东西收好，也会记得你说过的话。”
```

### 卸载准备选项

```text
【与阿雅郑重告别】山水有期，愿你寻得归处。
【暂不告别】此事以后再说。
```

### 卸载准备完成

```text
阿雅向太吾认真行了一礼。

“谢谢你带我走过这一程。”

“如果以后有人又被坏气息困住了，也请你告诉他：不要怕，总有人还记得他原来的样子。”
```

## 错误与异常反馈

### 后端返回错误

```text
阿雅怔了怔，似乎没有听清远处传来的声音。

“太吾，我有点看不清……这次先不要勉强，好吗？”
```

### 角色未注册或未找到

```text
风声掠过村口，却不见那名少女的身影。

此事似有差错，暂且不可继续。
```
